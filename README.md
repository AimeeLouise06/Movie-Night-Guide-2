# Movie-Night-Guide-2
The 'Movie Night Guide' is a simple, yet fun website where you can find recommending movies. Plan a fun night with your friends and family, choose a movie out of the recommendations and find the perfect snacks for you and the crew!

## Features:
- Curated movie recommendations by genre.
- Snack and drink pairing ideas.
- A simple, responsive web page to browse suggestions.

## How to use:
1. Clone or download this repository.
2. Open 'index.html' in your browser.
3. Explore more suggestions and plan your movie night!

## Technologies:
- HTML
- CSS3
- Markdown for documentation
